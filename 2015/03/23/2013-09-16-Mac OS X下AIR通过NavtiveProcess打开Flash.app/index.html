Mac OS X下，AIR通过NavtiveProcess打开Flash.app

AIR通过NativeProcess调用Terminal.app

让Flash.app打开指定的.jsfl

    AppleScript:test.scpt
    on run argv
    set jsflPath to item 1 of argv
    set appName to item 2 of argv
    set jsflFile to POSIX file jsflPath
    tell application appName to open jsflFile
    end run

这里用的POSIX File 你也可以用alias 只不过我嫌那个麻烦
如果是alias的话 就需要 ==HD:Users:{user}:Desktop:test.jsfl== 这样的路径

    Terminal command:
    osascript test.scpt \/Users\/{user}\/Desktop\/test.jsfl Adobe\ Flash\ CS5.5

{user}:看你系统里的用户目录名称

“/“以及空格之类的一定要转义

我是假定jsfl放在桌面 根据实情改变这个路径

再是我是flash cs5.5 这个也酌情修改～
