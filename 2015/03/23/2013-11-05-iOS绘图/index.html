iOS绘图

关于在iOS中绘图，目标是将每次从touchesBegan开始的笔迹绘制到不同的CALayer中，并且对该CALayer进行保存，即可实现类似PhotoShop一样的undo与redo，并且可以分图层设置样式。

好吧，上代码

    - (id)initWithFrame:(CGRect)frame
    {
        self = [super initWithFrame:frame];
        if (self) {
            [self setPath:[UIBezierPath bezierPath]];//设置路径
            [path setLineWidth:3.0];                  //设置线条粗细
            [path setLineCapStyle:kCGLineCapRound];  //设置线条未端样式
        }
        return self;
    }

    - (void)touchesBegan:(NSSet *)touches withEvent:(UIEvent *)event
    {
        CAShapeLayer* layer = [CAShapeLayer layer]; //创建一个新的CAShapeLayer
        [self setCurLayer:layer];                     //设置CAShapeLayer
        [self.layer addSublayer:self.curLayer];     //添加CAShapeLayer到显示列表中
        self.curLayer.delegate = self;
        [array addObject:self.curLayer];            //保存该CAShapeLayer到队列中
        [self.curLayer setFrame:self.frame];        //设置CAShapeLayer为当前视图尺寸

        layer.path = path.CGPath;                   //设置该CAShapeLayer所要绘制的路径，这样才能把从touchesBegan开始的所有点都绘制到新的layer上

        UITouch* touch = [touches anyObject];
        CGPoint p = [touch locationInView:self];
        [path moveToPoint:p];
        [self.curLayer setNeedsDisplay];
    }

    - (void)touchesMoved:(NSSet *)touches withEvent:(UIEvent *)event
    {
        UITouch* touch = [touches anyObject];
        CGPoint p = [touch locationInView:self];
        [path addLineToPoint:p];
        [self.curLayer setNeedsDisplay];
    }

    - (void)touchesEnded:(NSSet *)touches withEvent:(UIEvent *)event
    {
        UITouch* touch = [touches anyObject];
        CGPoint p = [touch locationInView:self];
        [path addLineToPoint:p];
        [self.curLayer setNeedsDisplay];
    }

    - (void)drawRect:(CGRect)rect
    {
        [[UIColor redColor] setStroke];               //当前图层中线条的颜色
        [path stroke];                            //填充当前图层中已绘制的线条
    }

1、上面的代码中没有undo/redo的功能，可以另外添加一个NSMutableArray，把每次新建的CAShapeLayer都存到这个数组中，
再添加一个UIButton，每次点击都删除NSMutableArray中最后一个CAShapeLayer并且[CAShapeLayer removeFromSuperlayer];

2、可以定义一个UIColor* curColor，每次设置一个新的颜色到curColor中，在drawRect方法中改成[self.curColor setStroke]

3、如果你设置了承载这些CALayer的UIView的backgroundColor为实色的话，每次新绘制的线条都会盖住上一次的，所以需要让CALayer的背景透明，就需要设置UIView的backgroundColor为[UIColor clearColor]

其它更多的效果大伙儿可以看一下官方的文档，可以为每个图层设置阴影、边框之类的样式，基本上就跟PhotoShop的图层类似了。
